# AEGIS 项目说明

目标:基于 AutoEncoder 的工业设备声纹异常检测,改进现有 AE 基线。
方法对比(最终产出一张跨2个数据集的对比表 + 消融):
- 对照组1:DCASE 官方 Dense-AE(dcase2023_task2_baseline_ae,不改,直接跑取基线)
- 对照组2:STgram-MFN(不改,直接跑)
- 实验组(我的方法 AEGIS):见下方分阶段

数据集:DCASE2020 Task2(数据集A)+ DCASE2024 Task2(数据集B),同一仓库切参数即可。
评估:AUC / pAUC / F1,6类机器各跑取平均,复用原仓库的评估代码。

实现纪律(重要):
- 不要改动原 baseline 代码;我的方法全部放在新目录 aegis/ 下,复用原仓库的数据加载和评估。
- 分阶段实现,每阶段先写 smoke test(1个机器类型、少量epoch)确认跑通,再继续。
  阶段1:Conv-AE(输入改成2D log-Mel谱图,Conv2D编码器+转置卷积解码器,MSE,异常分数=平均重构误差)
  阶段2:在阶段1上加频率轴注意力模块
  阶段3:加自监督分类辅助头,异常分数=重构误差+分类分数融合
- 每完成一个能跑通的阶段,用 git 提交一次。
